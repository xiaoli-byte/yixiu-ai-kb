## 上下文追问

1. 先问“这个客户当前商机阶段是什么？”，再问“下一步要做什么？”
2. 先问“哪个 SKU 活动价最低？”，再问“它还有库存吗？”
3. 先问“生日派对套餐包含什么？”，再问“需要提前多久预约？”
4. 先问“300 台 CIF 汉堡价格是多少？”，再问“如果换成鹿特丹呢？”
